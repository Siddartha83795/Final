# QuickBite Pro

A theme-aware food ordering system.

## Project info

A dual-location food ordering system for Medical Cafeteria and Bit Bites.

## Getting Started

### Prerequisites

- Node.js & npm installed

### IT Setup

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
